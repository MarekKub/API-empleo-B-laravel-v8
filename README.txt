 
OBJETIVO:
	El objetivo de la aplicación es crear un portal web que permita gestionar ofertas de trabajo
y demandantes de empleo.
	La web tendrá un sistema de login para usuarios. Estos podrán subir sus datos personales así
como su currículo, así como ver las diferentes ofertas de empleo, y postularse a ellas. Cada
oferta guardará los candidatos que se han apuntado para que luego el organismo o empresa pueda
revisar los currículos y elegir al candidato
	Hay dos tipos de usuarios principales: empleador y demandante. Hay un tercer rol que es el de 
administrador
	El rol empleador puede crear ofertas de trabajo, publicarlas, y editarlas, cambiar el estado 
de una oferta de trabajo ya publicada por él, ver los demandantes de empleo que se han apuntado
a la oferta, y mandar un email al demandante seleccionado.
	El rol demandante puede subir un currículo a su perfil, consultar las ofertas de trabajo,
apuntarse a ellas, y consultar el estado de las ofertas de trabajo a las que se ha apuntado.


TECNOLOGÍAS:

para ser compatible con servidores que trabajan con PHP 7.*.
